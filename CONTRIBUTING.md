## How to Contribute

1.  **Fork the repository** on GitHub.
2.  **Clone** your fork locally:
    ```bash
    git clone https://github.com/YOUR_USERNAME/GrindGuard-2.0.git
    ```
3.  **Create a branch** for your feature or fix:
    ```bash
    git checkout -b feature/amazing-feature
    ```
4.  **Commit** your changes:
    ```bash
    git commit -m "feat: Add amazing feature"
    ```
5.  **Push** to your fork:
    ```bash
    git push origin feature/amazing-feature
    ```
6.  **Open a Pull Request** on the main repository.

## Development

-   Run `npm install` to install dependencies.
-   Run `npm run dev` to start the local server.
-   Ensure all types are valid (`tsc --noEmit`).

## Code Style

-   We use **TypeScript** and **TailwindCSS**.
-   Please keep the UI "Cyberpunk/Minimalist" (Black backgrounds, Zinc borders).
-   Avoid adding heavy dependencies without discussion.

## Bug Reports

If you find a bug, please create an issue with:
1.  Your browser version.
2.  Steps to reproduce.
3.  Expected vs actual behavior.
